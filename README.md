# Cancer Diagnosis AI

## Overview
I built a neural network that classifies breast tumours as malignant or benign using the Wisconsin Breast Cancer Dataset. I built a 3-layer neural network (30, 15, and 1 neurons) with ReLU and sigmoid activations, with an 80/20 train/test split with StandardScaler normalization to ensure fair feature weighting. It achieved up to 99.12% classification accuracy on the test set.

## Data Source
Breast Cancer Wisconsin (Diagnostic) Data Set from here: https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data

## Result
Achieved up to 99.12% classification accuracy on the test set

## Tools Used
Python, TensorFlow, scikit-learn, pandas

## How to Run
Open cancer.ipynb in Jupyter Notebook or Google Colab. The dataset is available on the link provided. Create a directory "/content/tumour/" and place the data in there before running.
